# CreatiSignal

TikTok 电商素材增长决策台：从素材洞察、疲劳诊断到创意生成的一体化 Web 应用。

## Language

### 创意生成（Generate）

**创意生成（Generate mode）**:
创意助手中的一种模式，用户通过提示词与参考媒体生成视频/图片素材。
_Avoid_: 视频创作、AI 生成

**创意模板（Playbook）**:
一套可一键应用的完整创作方案：结构化提示词 + 参考媒体 + Hook/场景预设 + 推荐生成配置（痛点实测、UGC 测评、对比演示等），来源于 Top Ads 高表现广告的最佳实践。仅存在于视频生成类型。曾用名「创意打法」「模板」；代码标识符沿用 playbook/Template。
_Avoid_: 创意打法、预设、preset

**结构化提示词（Structured prompt）**:
应用创意模板后按【创意模板】【Hook】【场景】【产品展示】【卖点证明】【节奏与风格】【CTA】分段的多行提示词。标签是普通文本，用户可手改；【Hook】【场景】标签可点击跳转底部对应 tab。可一键「转自由文本」去掉标签。
_Avoid_: 表单、字段

**自由输入（Freeform）**:
未应用创意模板时的默认输入状态，用户完全手写提示词。

**引用（Reference token）**:
提示词文本中形如 `[Image #1]` / `[Video #1]` 的真实文本标记，指向引用行中对应编号的媒体。
_Avoid_: @提及、占位符、变量

**引用行（Reference row）**:
输入框下方的一行：上传槽位 + 引用胶囊。商品引用（lime 胶囊）固定在前，分隔线后是其他参考（图/视频，灰胶囊）；编号与引用 token 一一对应。
_Avoid_: 附件区、媒体列表、参考区（旧称）

**句段（Segment）**:
提示词中可被 Hook/场景选择器整体替换的句子（【Hook】【场景】标签后的整行内容）。用户手改过的句段无法精确匹配时，新句子降级为追加。

**Hook（钩子）**:
视频前 3 秒的开头句式（冲突、痛点、价格先出等）。可独立于创意模板点选，替换/填充提示词中的 Hook 句段。

**场景（Scene）**:
视频拍摄/呈现的情境设定（居家、户外、开箱、维修间等）。可独立于创意模板点选，替换/填充提示词中的场景句段。
_Avoid_: 场景化、背景

**Hook 变体（Hook variants）**:
生成 N 条时，每条使用不同 Hook 的变体策略；配置行「生成 N 条」旁的副文案。

**投放目标（Targeting）**:
平台 · 地区 · 语言 三元组（如 TikTok · 美国 · English），chips 行右侧可编辑，描述素材的投放语境。

**商品（Product）**:
用户自有的待推广商品，来源于商品链接解析或手动上传图片；其图片可作为引用进入引用行。
_Avoid_: 产品、SKU（指具体规格时才用）

**商品库（Product library）**:
已添加商品的集合，会话内持久，可跨生成任务复用。
_Avoid_: 商品列表、素材库（那是另一概念）

**链接解析（Link analysis）**:
粘贴商品链接后提取商品标题与图片的过程，UI 上表现为商品卡上的进度环。
_Avoid_: 爬取、抓取
