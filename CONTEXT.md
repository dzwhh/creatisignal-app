# CreatiSignal

TikTok 电商素材增长决策台：从素材洞察、疲劳诊断到创意生成的一体化 Web 应用。

## Language

### 创意生成（Generate）

**创意生成（Generate mode）**:
创意助手中的一种模式，用户通过提示词与参考媒体生成视频/图片素材。
_Avoid_: 视频创作、AI 生成

**模板（Template）**:
一套可一键应用的创作起点：分段提示词 + 参考媒体 + Hook/场景预设 + 推荐生成配置。仅存在于视频生成类型。
_Avoid_: 预设、样板、preset

**模板态（Template-applied state）**:
应用某个模板后的输入区状态：显示模板 chip，可切换或清除；清除后回到自由输入。文本始终可编辑——模板是起点，不是锁定表单。
_Avoid_: 模板模式、表单模式

**自由输入（Freeform）**:
未应用模板时的默认输入状态，用户完全手写提示词。

**引用（Reference token）**:
提示词文本中形如 `[Image #1]` / `[Video #1]` 的真实文本标记，指向参考区中对应编号的媒体。
_Avoid_: @提及、占位符、变量

**参考区（Reference tray）**:
输入框上方的编号缩略图区域，容纳参考图、参考视频、商品图；编号与引用一一对应。
_Avoid_: 附件区、媒体列表

**句段（Segment）**:
模板提示词中被结构化标记的句子（Hook 句、场景句、商品句、CTA 句），可被 Hook/场景选择器整体替换。

**Hook（钩子）**:
视频前 3 秒的开头句式（冲突、痛点、价格先出等）。模板态下可选可换，替换提示词中的 Hook 句段。

**场景（Scene）**:
视频拍摄/呈现的情境设定（居家、户外、开箱、维修间等）。模板态下可选可换，替换提示词中的场景句段。
_Avoid_: 场景化、背景

**商品（Product）**:
用户自有的待推广商品，来源于商品链接解析或手动上传图片；其图片可作为引用进入参考区。
_Avoid_: 产品、SKU（指具体规格时才用）

**商品库（Product library）**:
已添加商品的集合，会话内持久，可跨生成任务复用。
_Avoid_: 商品列表、素材库（那是另一概念）

**链接解析（Link analysis）**:
粘贴商品链接后提取商品标题与图片的过程，UI 上表现为商品卡上的进度环。
_Avoid_: 爬取、抓取
